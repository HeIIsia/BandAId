# BandAId
## Agentic AI for generating song lyrics in particular style

BandAid is a Python Tkinter app for generating original song lyrics based on a user’s chosen genre, mood, band influence, and text description. It can also load an image and use its visual atmosphere as part of the prompt, then display both the first draft and the revised final lyrics in a clean pop-up window.
